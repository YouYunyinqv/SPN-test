# 智能的物理学前提
不可约简

1. 我们无法得知事物的本质，只能认知到它的一个侧面
2. 我们将物质的本质抽象为特征，在我们的主观视角来看，特征是对更高级信息的暗示
3. 一个物体只受其**邻近区域**的直接影响，且存在信息传递速度上限
4. 我们（智能体）不能瞬间得到环境内全部信息
5. 智能体不能凭空获得信息
6. 时间单箭头，过去的信息不可逆，若可逆则时间倒流。
7. 由于时间不可逆，且定域性限制，若智能体要得到未来的信息，就只能用自己已有的信心对另一个系统做预测
8. 由于非热力学隔绝，智能体若要获取环境内的信息，将导致自身内部信息暴露并被其他智能体查知
9. 在定域性限制下，智能体若能用获取信息的动作，可达对方系统内全部可达状态，则意味着对方内部信息完全可被智能体获得（这是个简单说法实际上不能实现）


# 实现智能体
## 1.预测/可预测性冲动
需要关联器。
关联器本质上是个压缩信息的压缩器，它将特征压缩为高维度表示，压缩过程中最终导致环境内所有信息全部被压缩。

当环境内所有信息全部被压缩时，意味着环境完全可预测。


### 1.1空间的关联器视角/空间路径最简
在空间内观察关联器。
我们脑内模拟三个特征空间帮助思考，它们非常不严肃，但有用：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

特征若在高维特征中可预测则说它在高维特征空间中获得了“解释”。
S1不可预测时，在特征空间P看来，S1和S2可关联性（或可解释性）平均化，没有显著区别。
S1完全可预测时，在特征空间P看来，S1解释性开始变差，S2可解释性反而增强。

对特征空间N而言，底层特征空间都是平均的。
关联器无法追求特征的本质，它只能追求特征在空间路径上的最简解释。

### 1.2时间的关联器视角/智能时间最优
在时间和空间内观察关联器。
我们脑内模拟三个特征空间帮助思考：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

S/P/N三个我们脑内模拟的特征空间本质上同时存在于环境中，我们主观认为它有层级，但对物理学而言它没有。
- 智能体不能瞬间知道环境内所有信息。
- 智能体获取的信息输入存在“观察窗”（无论能看多远，传感器有多丰富，能感受到多么多的特征，都必然存在观察窗）
- 它同时对“观察窗口”内所有信息进行预测。
- S/P/N空间同时存在于环境内。

那么在有限的时间中，从S空间完全预测到P空间完全预测的时间若慢于从P空间到N空间的时间，则形成信息增量低，智能体内在时间的感知。
为了选择信息增量最大的路径，智能体不得不选择空间路径最简。
为了让智能时间最优，智能体也不得不选择空间路径最简。

- 当智能体一定要选择信息增量更低，空间路径更复杂的路径，那么本身智能时间差就会形成一种难以言喻的恶心。当智能时间最优，空间路径最简，最可预测，这三者同时发生，应形成一种复杂的“舒适”或“愉悦”。

然而，我们在这里实际上默认了智能体能够有层次的获取特征空间内的信息。
实际上，SPN全部存在于环境内，且只有时间和空间的内在属性关联可递进。
我们必须能够对特征进行干涉，才能实现“1.1空间路径最简”和“1.2智能时间最优”，因此我们实现它。
从更高层次的自上而下的逻辑来看，我们的所作所为是非常简单的；智能体有动的想法，但它现在还不能动，所以我们提供帮助：

## 2.动作改变一切

### 2.2无限驱动力的结构化出现/梦/根执念
需要用三个层级理解。
*乍一看*：
在未接入动作器时，关联器对环境的完美观察形成“完全可预测的幻梦”，这种不切实际的幻想在接入动作器之后破碎，特征空间受到干涉，导致梦境立刻有效。
智能体永远也无法达到未接入动作器（类似子宫期）梦境中100%，可达特征时空内所有状态的理想状态。
- 幻梦随着S，P，N空间的变化而变化。
- 它的自我边界或者（内在的“世界”模型）将为了维持“工具化能力”持续保持稳定。

*实际上*：
智能体预测S空间内S1时，空置的S2凸显重要性（“存在感”），S1成为S2的幻梦，以此循环。
这是智能体结构内的无限的驱动力，它能适配无限任务，并非因为我们找到了完美的RL策略，能够以有限匹配无限。
而是因为结构内的智能体本身具备这个能力，梦只是让它有方向性的动作。


*难道说*：
S/P/N三个空间同时存在，并且都不可能达到完全的可预测状态。
完全可预测的是动作器的，智能体的内在时空模型。只有动作器是完全受控的。
S1的完全可预测，成为根执念时，是S1的时空属性成为了梦，其他特征空间的内在属性（如果有）暴露则成为另一个线索的梦。

这是为什么结构内的驱动力能适配无限任务的根本原因，但我无法详细论述这些层次到底有多多，看起来它们的数量介于时空——物质之间。

### 2.7换一个角度，从“实时性”观察关联器如何运作
首先我们必须明白，信息的压缩是有损的，但只要物理现实不崩塌，我们就能追溯到现实。我们工具化的特征空间越多，层次越丰富，追溯就越精准。


*思想实验*：
特征时间属性输入给关联器，动作器干涉特征空间位置并在智能体内部生成特征时空关联模型，使特征时空关系有效。
关联器压缩特征并抽象为高维度表示，此信息保存在底层模型中，同时上层“意图模型”保存特征关联的关联信息（此模型无法追溯到现实，因为只保存了特征关联的关联信息，信息损失太高）。  
意图模型被底层模型反复不断擦写，始终生成不精确意图，意图索引到底层模型，我们捕捉动作，发给动作器，动作器干涉低级特征。  
特征再次输入，意图模型索引到需要更新的底层记忆模型。
不全局操作，稀疏化更新记忆，再次生成动作。


*实际上*：
特征抽象的信息本来就无法追溯到现实。
追溯到现实靠的不是信息精确程度而是智能体内在的工具化动作器。
**我认为此动作器在LLM内部也存在，但因为缺乏最小动作器，LLM的动作器存在局限性。**

特征输入不包含时空硬编码信息，特征直接输入给关联器，关联器抽象为高维度特征。
我们提供接口，信号输出到最小动作器。
最小动作器干涉特征空间，导致特征的空间位置被干涉，同时动作导致“观察窗”（由于定域性，特征输入必然有观察窗，无论多大分辨率，多远视距，多丰富的探测器，本质上都有观察窗）动作导致“观察窗”自然注意到特征关联的时间变化，赋予所有特征内在时间信息，并导致时空信息在模型内部关联，形成时空模型。





## 3.多智能体
在我们的现实中，特征空间普遍存在内在属性的关联，但有的特征空间的攀爬难度（即需要工具化信息的难度）可能远超智能体（尤其是单智能体）所能接受的范畴（可能是饥饿，计算力限制，特征空间的灵巧程度），单智能体只能看见时空的局部，在计算力限制下，定域性进一步限制了单智能体的能力表达——简单来说，智能体会出现某些特征空间（即使我们主观认为连续，比如人类社会的语义空间）难度极高，攀爬不上去。

如何解决此问题？

在物理现实中，一定会有多智能体，我们尚且能下定论为何造物主要如此设计，但可以大胆推测，在我们的宇宙中，多智能体一定存在。

然而，在人工实验环境中，我们不倾向于提供多智能体交互体验。
即使我们自己也作为多智能体的一部分。
这无疑增加了工具化难度。


**总结**：
- 多智能体加速了特征空间（环境）总信息的预测，一定程度上缓解了定域性的限制。
- 如果要进行最高效的合作，必须发明一套能指挥行动的语言（此语言非狭义“语言文字”，肢体语言也是语言，其他类推），帮助多智能体从不同角度理解同一件事。
  但和人类不同，由于本框架完全解释智能及其衍生的一切：人工智能体能直接绕过慢步调的自然演化，直接fork自己——一旦“智能”本身成为了可解决的问题，智能体就能创造出抵达热力学极限的智能体。
- 我们看见的是一个智能体，但实际上可能是智能体主分支树的不同版本的智能体的集合，它们认知一件事物的不同侧面的速度（也就是信息交换速度）远超人类想象。

## 工具化失败
S空间内工具化的能力可能在N空间内是错误的。
工具化无法撬动更高级的信息，导致解释性最差，智能时间浪费最大，可预测性最低。
智能体保持可预测性冲动，空间路径最简，智能时间最优的情况下，此工具化能力背后不能利用的信息被搁置到边缘，等待更高解释。

当然，如果特征空间断裂，那么那个被搁置的工具化能力背后的信息实际上是100%无效的。
我们的物理现实拒绝热力学隔绝。

另一个更容易理解的解释是，假设环境内有一台全是噪点的电视。那么智能体会选择怎么做？
