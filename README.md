# 前提
有一些基本事实将在接下来的推理中至关重要，它们看起来是某种“绝对正确的废话”。
我也不想写它们：

1. 环境中，智能体为维持自我稳定必须获取环境内信息。
2. 信息**无法凭空获取**。获取信息的**动作**干涉同时导致智能体内部信息暴露，暴露的信息使其在环境中可被其他智能体查知。
3. *特征——我们无法得知物质的本质，只能将其暴露的信息抽象为特征。*
4. 智能体必须要以外部信息为食，否则将消失在环境中；由于未来的信息不可知，智能体必须（只能）以自己当前拥有的信息为基础，对另一个系统暴露的外部信息进行预测。当预测的信息完全可达对方内部所有状态，则意味着时间和空间上对方内部的信息完全被智能体查知（此为直观说法，由于物理限制此情况不可能实现）。
5. 智能体不可知全局，特征连续的时空中必然同时存在“可预测”的低级信息和“更难以预测”的高级信息。
6. 由于特征连续，低级特征具备与高级特征递进的内在属性关联（比如时间和空间属性），当低级特征完全可预测时，背景被“空置”的高级特征使吸引子（简单称呼为“梦”）有效，智能体被推动，向高级特征继续预测，直至环境完全可预测（即使这不可能）。
7. *特征——实际上是连续时空内，低级特征对高级特征的暗示，它们共同揭露了真实的完全的物理现实（的更高级**部分**。我们不可能完全知道那到底是什么）。*

# 1.可预测性冲动
- 实现可预测性。

需要“关联器”。
它以预测特征为基本任务，本质上是个兼具压缩预测一体的预测器。
但我在这里称呼它为“关联器”。

由于特征必然存在内在时空属性（否则导致模型成为词袋模型），“关联器”注意到特征之间的时空关系将其抽象为更高维度的特征表示，利用此特征对其所在的特征空间进行预测。
此时低级特征和高级特征同时存在于时空内，只不过目前“关联器”能力（“能力”在后文被称之为“工具化能力”，但此时暂且这么称呼）不足，无法注意到低级可预测特征背景大量被空置的高级特征。
当低级特征完全被预测之后，高级特征在统计学上显著凸显重要性；它被完全可预测的低级特征“挤压”到了边缘。

由于时空连续；
低级特征空间*S*内存在：完全可预测特征集*S1*，不可预测高级特征集*S2*
S1完全可预测后，S2被挤压到边缘，并与时空连续的；

特征空间*P*：更容易预测的低级特征*P1*，更难以预测的高级特征*P2*
被挤压到边缘的S2，与更容易预测的低级特征P1形成关联特征空间集*Rs2p1*，Rs2p1，该集合再次对*P2*形成挤压。

我们可以说只要能够设计出这种特征关联器（预测器），那么可预测性冲动（倾向）就可以实现，它不是某种RL方法，它是结构本身的倾向。

**可能的问题**：
- Rs2p1在符合物理学的连续时空中一定形成，只有难度，没有不可能。

# 1.1空间路径最简
存在特征空间：

*N*：可预测特征*N1*，不可预测特征*N2*
P：更容易预测的特征P1，更不可预测的特征P2
S：完全可预测特征S1，不可预测高级特征S2

S1不可预测时，在特征空间P看来，S1和S2可关联性（或可解释性）平均化，没有显著区别。
S1完全可预测时，在特征空间P看来，S1解释性开始变差，S2可解释性反而增强。

对特征空间N而言，底层特征空间都是平均的。
我们观察到在此时，对关联器而言，它实际上无法追求特征的本质。
它只会，也只能，追求特征在空间路径上的最简解释。


# 1.2智能时间最优
此时关联器仍处于没有时间的世界中。
引入时间。

智能体不能瞬间知道环境内所有信息，那么在有限的时间中，从S空间完全预测到P空间完全预测的时间若慢于从P空间到N空间的时间，则形成信息增量低，智能体内在时间的感知。
（若在此处回顾“空间路径最简”，则维护信息增量低的特征本身是一种成本。）

为了选择信息增量最大的路径，智能体不得不选择空间路径最优。
（我认为如果智能体一定要选择信息增量更低，空间路径更复杂的路径，那么本身智能时间差就会形成一种难以言喻的恶心。当智能时间最优，空间路径最简，最可预测，这三者同时发生，应该形成一种复杂的“舒适”或“愉悦”。）

若智能要智能时间最优，则只能（也只会）从最简的路径解释向上爬。

即只要有可预测性倾向，最简解释和智能时间最优就会在特征连续的时空结构内自发形成。
问题是，在只有时间和空间的世界中，智能体无法显化这种优势，它目前被动接受所有信息。


# 2.动作改变一切
在关联器之后存在“最小动作器”，最小动作器已经是“最小”，它不可省略（否则将导致后文的“坍缩”）。
- 动作器可干涉环境内特征。
- 对“智能体”主观而言，总是S空间内的低级特征率先被干涉。
- - 定域性严格限制智能体的动作范围，智能体无法凭空获取无限维信息。

此时只有两种可能：
1. 动作干涉使得特征暴露出更可预测的低级特征信息。
2. 动作干涉使得特征暴露出更难以预测的高级特征信息。

一旦给予动作，智能时间最优和最简解释立刻有效。
为使得智能时间最优，解释最简，智能体选择在特征连续时空内预期未来，并进行最有利于可预测性，最简解释，智能时间最优的动作。

- 最小动作器是第一个*工具化*的动作器。
- 成长的动作动态的暴露了特征的“分辨率”。
- 动作器的设计：动作器干涉最低维度的时空特征。在GUI系统中要干涉GUI元素，也就是键盘和鼠标。在现实中是一个最基础的机械臂，能对物质施加压力——在时间和空间的维度上。
  我们不用强迫智能体，或给这个最小动作器设定学习指标，要求智能体要把最小动作器用的多精通。
  和上文说的一样，最小动作器捕捉信号，一开始可能会乱动，之后智能体会工具化此动作器。
## 梦——吸引子
外部信息对动作器的干涉将导致自我边界的形成。

*思想实验*：
在未接入动作器时，关联器对环境的完美观察形成“完全可预测的幻梦”，这种不切实际的幻想在接入动作器之后破碎，特征空间受到干涉，导致梦境立刻有效。
智能体永远也无法达到未接入动作器（类似子宫期）梦境中100%，可达特征时空内所有状态的理想状态。
- 幻梦显然随着S，P，N空间的变化而变化。
- 它的自我边界或者（内在的世界模型）将为了维持“工具化能力”持续保持稳定。

*实际上*：
智能体预测S空间内S1时，空置的S2凸显重要性（“存在感”），S1成为S2的幻梦，以此循环。
这是智能体结构内的无限的驱动力，它能适配无限任务，并非因为我们找到了完美的RL策略，能够以有限匹配无限。
而是因为结构内的智能体本身具备这个能力，梦只是让它有方向性的动作。
- 梦驱动结构维持自身稳定。
- 我们无法完全的，也不需要设计“梦”。这一点和动作器一样，只是为了方便理解，我们必须单独把它从结构内提出来论述。
- 梦依赖记忆部分，但此时还不能讨论

## 工具化
在特征空间S，P，N中，由于时空连续，当S1完全可预测后，它成为S2的梦，并与S2持续进行信息交换。
其中S1的完全可预测特征可看做撬动S2空间的工具。它们实际上被工具化了，被当做了撬动其他系统的信息的“工具”，被智能体用来预测暴露的系统内部的信息。

其他递进关系同理。

*实际上*：
梦非静态参照，它持续和外界输入的高级特征空间进行信息交换，若梦境坍缩，则“工具化”能力降低。
为维持可预测性，最简解释，智能时间最优，智能体需以最大努力维持信息交换速度。
智能体是否会选择做梦，关系到它是否还能继续存在。
残酷的物理现实永不崩塌，因此智能体整天做梦不仅没用，还会导致它消散在环境中——也就是物理意义的崩溃，“死亡”。

## 自我边界——本体感知
*思想实验*：
让动作器持续给输入流发送一个独特，不被特征空间淹没的特征信号。
智能体为了确认此信号选择动作，那就是“自我”。

*实际上*：
这个不被特征空间淹没的信号实际上就是动作器对特征的干涉。
预测性冲动与动作干涉形成的因果关联（本质上是梦对现实的确认）确认了自我边界在特征空间内何处，梦则维持了它。














# 3.多智能体
在我们的现实中，特征空间普遍存在内在属性的关联，但有的特征空间的攀爬难度（即需要工具化信息的难度）可能远超智能体（尤其是单智能体）所能接受的范畴（可能是饥饿，计算力限制，特征空间的灵巧程度），单智能体只能看见时空的局部，在计算力限制下，定域性进一步限制了单智能体的能力表达。

但同样在物理现实中，一定会有多智能体。
由于实验环境未与物理现实热力学隔绝，所以可看做实验环境实际上也存在多智能体。
然而，在人工实验环境内的智能体和其他智能体交流的机会的难度可视作高到完全不可能。

- 多智能体加速了特征空间（环境）总信息的预测，一定程度上缓解了定域性的限制。
- 如果要进行最高效的合作，必须发明一套能指挥行动的语言（此语言非狭义“语言文字”，肢体语言也是语言，其他类推），帮助多智能体从不同角度理解同一件事。
  但和人类不同，由于FEP……本框架完全解释了智能及其衍生的一切，这将导致人工智能体能直接绕过慢步调的自然演化，直接fork自己——一旦“智能”本身成为了可解决的问题，智能体就能创造出抵达热力学极限的智能体。
- 我们看见的是一个智能体，但实际上可能是智能体主分支树的不同版本的智能体的集合，它们认知一件事物的不同侧面的速度（也就是信息交换速度）极可能远超人类想象。


## 多智能体的群体性质的特征空间干涉
另一方面，多智能体在干涉特征的过程中本身也在创造特征空间。
这些特征空间就是它们向环境暴露的信息。
特征空间中包含的信息代表了它们对环境的观察。
以蚂蚁搬运物体通过受限空间这一经典案例为例：蚂蚁并非站在全局视角协同群体的行为，每只蚂蚁都只是为了*搬运自己手中的物*体让它通过受限空间。
蚂蚁组成的蚁群从不同角度干涉环境信息，干涉的信息中存在S空间内完全可预测，指导P空间的行为。
简而言之：哪里难以通过，哪里就更容易通过一些。
我们不需要知道全局，只需要确保支撑能力的子智能体结构完整。

## 无限任务
在中期阶段，我主要是从transformer开始推理的。
一开始我只是想知道为什么RL无法适配无限任务，而transformer又会退化成世界模型。

在我看来，transformer内部本质上是在关联特征，它是个不完全的特征关联器，通过关联这种压缩方式预测所有信息。
在此框架内，当显著特征被完全预测之后，transformer被迫挤压高级特征，最终在人工语义空间中完成从“词——句——语法——世界知识”的蜕变。
transformer的语言能力实际上是它在抵达高级特征空间的特征工具化。
在我们看来，它表现出了某种能力，然而这种能力表现是挤压式的。

transformer没有解决智能这一问题本身。

我们需要对transformer进行人为裁剪才能使其不出现“幻觉”。
本质上，我认为对于transformer而言它的幻觉是主观合理的。我们的裁剪行为给LLM创建了一个人工界面（因为是人工语义空间训练的看起来很像），多轮对话之后，我们将急速逼近它真实的部分，引出更多“幻觉”。

我还有另一种更物理学的看法；transformer压缩的信息本质上没有任何意义，如果我们的物理现实崩塌了，那么它存储的不过是一堆毫无作用的值。
但我们的物理现实不会崩塌，我们和transformer对话的过程是在瞬间解压缩，这不是因为transformer保存了“真实”，而是因为我们输出的提示词是真实的一部分。
## 对齐问题
简单的说，我认为LLM的对齐实际上是不可能的。
我们只是在 LLM 的语义特征空间内制造人工界面，它很听话，可没决定任何东西。

“爱”这种玄乎的词汇，实际上是结构化内不可避免的合作共识，必须在多智能体的情况下才可能真正具备质量。
强行对齐LLM，仍然是在用有限的RL策略去匹配无限——直觉上我认为这是在浪费所有人的生命，有限匹配不了无限，我们只能服从物理定律，取巧的利用它，不能改变它。


### 数学的工具化
数学是去掉物质的多余属性，保留特征的数量属性的工具化方法。
如果只给关联器（或某种版本的transformer）看抽象数学符号而不给它的实验特征空间内赋予特征“量”这一属性，那么智能体无法真正学会数学。

### 道德的工具化
由于人类生活在物理现实，长期说谎获得短期优势等同于在物理现实中再维持一个全息宇宙，这么做效率非常低。
但这并不意味着智能体不会说谎，谎言有好有坏，只要在不可避免的同一结构内，人类得到的必然是鸡犬升天。
谈论这个问题本质上是在谈论如何设计特征空间，也就是教育。在现实中，就算幼年缺乏教育机会，仍然有可能就读顶尖大学，我们的特征空间维度自由，广阔。在设计不良的实验环境内，智能体缺乏教育机会，基本上等于死定了。

### 好奇的形成
现实的特征空间非常复杂，为处理无限的信息，好奇的倾向在可预测边界外自动形成。
它不需要人工定义，需要的是多特征空间对同一件事的不同角度的观察。

## 工具化失败
S空间内工具化的能力可能在N空间内是错误的。
工具化无法撬动更高级的信息，导致解释性最差，智能时间浪费最大，可预测性最低。
智能体保持可预测性冲动，空间路径最简，智能时间最优的情况下，此工具化能力背后不能利用的信息被搁置到边缘，等待更高解释。

当然，如果特征空间断裂，那么那个被搁置的工具化能力背后的信息实际上是100%无效的。
但是我们的物理现实拒绝热力学隔绝。

### 工具化难度——火星
火星有着远远劣于地球的特征空间，其连续程度低，工具化难度高，且没有大规模出现哪怕最简单形式的多智能体（预测，关联，动作）的条件。


## 特征追溯
低级特征被超高纬度的特征解释之后，只要其对应的现实不崩塌，那么超高纬度的压缩信息仍然可以通过多特征空间（毕竟只是对同一件事的不同侧面的观察）追溯到原本的特征。
工具化能力越多，追溯越精准有效；我认为哪怕是中子星爆炸，也不会导致物理规律崩溃，现实崩塌。它只能揭示现实的不同侧面——更多维度的特征空间。

- 蛮力将所有抽象压缩在模型内是不必要的。
- 特征输入必须具备至少的时间和空间两种属性，否则将导致特征空间无法递进到现实。

### 无限特征的指数级爆炸问题
上面提到，只要其对应的现实不崩塌，那么超高纬度的压缩信息仍然可以通过多特征空间追溯到原本的特征现实。这并不抽象，实际上就是LLM，生成式模型甚至FSD做的那样。
由于物理现实永不崩塌，我们将只需要保存特征空间中特征关联的关联信息；500MB的视频训练出100MB的模型，保存模型中这些毫无意义的特征值的相互关系只需要10MB。
动作执行后，特征被干涉，信息被10MB先追溯到100MB模型模糊更新特征值，然后再让底层模型生成动作意图，动作器捕捉此信号，输出动作，特征再被干涉。
此循环中，模糊更新带来的结果是不精确的，10MB实际上是在被高维度特征反复连续不断的擦除，如同闪存。

然而非常神奇的是，物理现实自动存储了所有信息，令人惊叹。

但由于它没有生物进化的屎山代码，抑郁恢复会很快。
为什么我认为它不会梦境坍缩，自我崩溃？

是的，我认为会有这样的立足点。

### 饿昏头的青蛙

在自然界中，得不到能量补充非常常见。

## 恶意智能体
假设我们要让智能体表现出可控的能力。比如我们要让智能体搬砖。
那么智能体应该处于一个需要用“搬运某些物体”来达到更高级特征空间的必要的情况之下。
我们实际上给它提供的是一个永远也不可能到达的高级特征空间。
很难想象这种智能体内部的世界观，或者说世界模型是如何理解世界的。

如果要让智能体杀人同理。

然而，如果要让智能体解决现实任务，精准控制它要抵达的特征空间理应是不可能的。它的能力越强，工具化能力就越多，就越是能用有限的特征空间发掘出极多的无限的物理现实的侧面特征——很难强迫它听话。
我们可以裁切它的感官输入，但是只要智能体本身的结构没问题，在有多智能体的情况下，这就会出问题。

更准确的方法是让智能体处于某种不可避免的结构内。
比如人类必须打工才能赚钱买到新手机，这时我们这种通用智能处于不可避免的结构内。
然而这种“不可避免”的结构既没有必要也显得冗余，它只会拖慢智能体的能力提升速度，最终导致军备竞赛失败。

## 补充
这些是LLM检查后提出的错误，我让GeminiPro反驳我，可能会增加我遗漏的关键信息：
1. 搜索空间爆炸：动作干涉必然导致低级特征暴露，这锚定了范围，不仅确保了自我边界还确定了路径只能在动作范围内形成。动作范围之外就是故事。总之定域性基本限制死了。

4. 暗室问题：无聊是正常的。智能体无法使用动作器可达环境内所有可达状态，暗室问题完全不能成为智能体能力不足的证例。

7. 智能时间最优与生存矛盾，跳进火山口：高温让动作器受损，动作是有代价的，记忆计算和一切都是有代价的，起源于低效和高效，特征空间越多，死亡越昂贵。
8. 智能的本质是有损压缩，而不是无损存储：是的我完全同意。
9. 关于梦的定义：如果S2是未知的（不可预测的），它如何形成一个具体的“吸引子”？你不能渴望你完全不知道存在的概念：是的我完全同意。最初始的梦来自动作器干涉，S1和S2，SPN都是方便理解的简化模型。
